## Installation



 Install dependencies:

    ```bash
    pip install -r requirements.txt
    ```

## Usage

1. Run the Django development server:

    ```bash
    python manage.py runserver
    ```

2. Access the website through your browser at [http://localhost:8000](http://localhost:8000).
